# [abhe.in](https://abhe.in)

## About

My personal website—a clean, minimalist landing page and digital resume. Originally an experiment in building a custom Static Site Generator from scratch in Python ("Monochrome-SSG"), it has since been streamlined and transitioned to Jekyll to focus entirely on fast, reliable, and straightforward content delivery.

## Built Using 

* [Jekyll](https://jekyllrb.com/): Static site generation
* [HTML](https://en.wikipedia.org/wiki/HTML) & [CSS](https://en.wikipedia.org/wiki/CSS)
* [Git](https://git-scm.com/): Version control
* [GitHub Pages](https://pages.github.com/): Hosting

## Objective

This repository serves strictly as:
* The source code for my professional resume and landing page.
* A clean, easily maintainable hub linking to my other engineering and writing projects.

## Rationale

While building a custom Python SSG was a valuable learning exercise in understanding the mechanics of web generation, managing the technical debt detracted from my primary goals. Moving to a standard, natively supported generator like Jekyll provides layout inheritance out-of-the-box and aligns perfectly with GitHub Pages. This simplified workflow allows me to focus my programming efforts on more complex data engineering and AI projects, rather than website infrastructure.

* Conducting deep-dive competitive analyses (e.g., the Wood Logging Industry) and investment due diligence.
